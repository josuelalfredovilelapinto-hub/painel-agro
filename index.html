Memorial de Cálculo e Equações Internas (v10.8)

Documentação matemática dos algoritmos executados no navegador em tempo real.

1. Módulo de Psicrometria e Dinâmica Atmosférica

Temperatura do Ponto de Orvalho (Magnus-Tetens OMM):

Tpo = T - ((100 - UR) / 5)

Diferencial Térmico de Evaporação (Delta T):

DT = T - Tpo
• DT < 2 ou DT > 8 → [STOP] | • DT entre 2 e 8 → [IDEAL]

Déficit de Pressão de Vapor (Equações de Tetens):

Es = 0.6108 * exp((17.27 * T) / (T + 237.3)) [Pressão de Saturação, kPa]
DPV = Es * (1 - (UR / 100)) [kPa]

2. Módulo de Evapotranspiração e Balanço Hídrico

Estimativa de Radiação Solar Global Sem Piranômetro (Curva Cosseno Adaptada):

Se hora entre 6h e 18h → Rg = 800 * sin(π * (Tempo_Decimal - 6) / 12) | Noite → Rg = 0

Evapotranspiração de Referência Horária (Makkink Simplificado):

ETo = 0.7 * (Rg / 694.4) * (T / (T + 237.3)) [mm/h]

Contabilidade de Massa Hidrológica do Solo (CAD Solo Argiloso - Perfil 40cm):

CAD_atual = CAD_anterior + Chuva_atual - (ETo * Δt)
Restrições de Contorno: Max_CAD = 60.0 mm | Min_CAD = 0.0 mm

3. Módulo de Mecânica Celeste e Biometeorologia

Declinação Solar (δ) e Fotoperíodo Teórico (N) para Laranjeiras do Sul (Lat -25.4125°):

δ = 23.45 * sin((360 * (284 + DOY) / 365) * π / 180)
cos_ws = -tan(-25.4125 * π / 180) * tan(δ * π / 180)
N = (2 * acos(cos_ws) * 180) / (π * 15)

Acúmulo Térmico Biológico (Graus-Dia - Tb 10°C):

GD = max(0, T - 10)

4. Modelos de Favorabilidade de Pragas (Eco-Sistemas)

A. Cigarrinha-do-Milho (Dalbulus maidis):

• Se T < 15°C → Risco = 5% (Paralisia metabólica por frio)
• Se 25°C ≤ T ≤ 32°C → fT = 0.8 | Outras faixas → fT = 0.4 ou 0.3
• fU = (UR < 65%) ? 0.2 : 0.05 (Ar seco favorece voo migratório)
• Risco_Bruto = (fT + fU) * 100
• Se Chuva > 2mm → Reduz para 30% do valor (Lavagem mecânica de ninfas)
• Se Fora da Janela ZARC (Ponte Verde ativa) → Risco = Risco + 15%

B. Percevejo-Marrom (Euschistus heros):

• Se 26°C ≤ T ≤ 30°C → fT = 0.75 | Outras faixas → fT = 0.4
• fU = (UR entre 50% e 70%) ? 0.15 : 0.05 (Umidade moderada otimiza ataque a vagens)
• Risco_Bruto = (fT + fU) * 100
• Se Chuva > 5mm → Reduz para 50% do valor (Mortalidade de ninfas instar 1 e 2)

C. Lagarta-do-Cartucho (Spodoptera frugiperda):

• Se 25°C ≤ T ≤ 33°C → fT = 0.8 | Outras faixas → fT = 0.3
• fU = (UR < 75%) ? 0.15 : 0.05 (Noites quentes/secas favorecem postura da mariposa)
• Risco_Bruto = (fT + fU) * 100
• Se Chuva > 4mm → Reduz para 40% do valor (Afogamento larval no cartucho)

D. Lagarta Falsa-Medideira (Chrysodeixis includens):

• Se 22°C ≤ T ≤ 28°C → fT = 0.7 | Outras faixas → fT = 0.3 (Prefere climas mais amenos)
• fU = (UR ≥ 60%) ? 0.2 : 0.05 (Comum no microclima úmido do baixeiro da soja)
• Risco_Bruto = (fT + fU) * 100
• Se Chuva > 6mm → Reduz para 50% do valor

5. Modelos de Favorabilidade de Doenças

Ferrugem Asiática (Phakopsora pachyrhizi):

Se UR > 50% → Risco = [Se 18°C ≤ T ≤ 26°C → 1.0, se não → 0.4] * ((UR - 50) / 50) * 100

Mofo Branco (Sclerotinia sclerotiorum):

Se UR > 60% → Risco = [Se 15°C ≤ T ≤ 21°C → 1.0, se não → 0.3] * ((UR - 60) / 40) * 100
